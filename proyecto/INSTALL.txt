INSTALL
-------


Para el funcionamiento de este programa se necesita lo siguiente:
	Python 2.7 
	Numpy
	PIL
	OpenCV

Puedes intalarlo de la siguiente manera cada uno. 

Python 2.7- Linux
------------------

Python algunas veces puede venir pre-instalado en sistemas operativos Linux(Ubuntu), en caso de no estar instalado pueden descargar el source desde: 

http://www.python.org/getit/

----------------------------

Python 2.7 - Windows
----------------------------

Para instalarser Python en Windows, se puede descargar el instalador
desde el enlace:

http://www.python.org/getit/

Se debe descargar la versión 2.7.4 que es la mas actual, se ejecuta el instalador y seguir los pasos indicados por este. 
-------------------------
NumPy y SciPy
------
Descargarlo el instalador desde este enlace:
http://www.scipy.org/Download

En Ubuntu, ejecutar el comando:

sudo python setup.py install 

-------------------------

OpenCV - Windows
================

Descargar el instalador en el siguiente enlace:

http://opencv.willowgarage.com/wiki/InstallGuide

Ejecutar el instalador y seguir las instrucciones que sean indicados.

OpenCV - Python
===============

En Ubuntu, ejecutar el comando:

sudo apt-get install python-opencv

Ingresar la contraseña cuando se pida.
